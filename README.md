Overview

A fully-featured E-commerce web application built with React, TypeScript, and Redux Toolkit. This project emphasizes clean code, performance optimization, reusable components, and modern frontend best practices.

🌟 Key Features

Dynamic Product Listings: View products with categories, filtering, and search functionality.

Shopping Cart: Add, remove, and update product quantities with Redux Persist for saved state.

Custom Hooks: Encapsulated logic for reusable, clean, and maintainable code.

Performance Optimizations:

Skeleton loaders for smoother user experience

Efficient state management with Redux Toolkit

Lazy loading and code splitting where needed

🧩 Tech Stack

Frontend: React, TypeScript, Tailwind CSS

State Management: Redux Toolkit, Redux Persist

Routing: React Router

API: RESTful API integration

Code Quality: Clean code principles, reusable components, modular structure

UI/UX: Responsive design, skeleton loaders, smooth interactions

⚡ Highlights

Emphasis on Clean Code with reusable and modular components

Performance-focused, with optimized rendering and state management

Enhanced user experience using skeleton loaders for data fetching

Fully responsive layout supporting all device sizes
